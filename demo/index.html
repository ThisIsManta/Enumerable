Array.prototype.toObject
code

Returns an object by constructing properties from all members.

['a', 'book'].toObject();
// { "a": true, "book": true }

['a', 'book'].toObject(function (x) {
  return x.length;
});
// { "1": "a", "4": "book" }

Array.prototype.toString
code

Returns a string by concatenating all members. This presumes all members are string. ['La', 'Da', 'Dee'].toString();
// "La,Da,Dee"

['La', 'Da', 'Dee'].toString('-');
// "La-Da-Dee"

See also Array.prototype.join
